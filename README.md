# TaskFlow AI - Premium Project Management Platform

A modern, full-stack SaaS project management application built with React, Node.js, and MongoDB.

## 🚀 Features

- ✅ User Authentication (JWT)
- ✅ Project Management
- ✅ Task Management with Kanban Board
- ✅ Team Collaboration
- ✅ Role-Based Access Control
- ✅ Real-time Notifications
- ✅ Beautiful Modern UI
- ✅ Responsive Design

## 📋 Tech Stack

**Frontend:**
- React 18 + Vite
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide Icons

**Backend:**
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Bcrypt Password Hashing
